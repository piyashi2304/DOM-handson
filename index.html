In the context of web development and browser scripting "document" and "window" refer to the different objects that are part of the Document Object Model (DOM) and represent different aspect of a web page.Here's the difference between them:

    Window
  1. The Window object represent the browser window or tab in which the web page is loaded
  2. It serves as the global object in the browser's Javascript environment and provides methods and properties for controlling the browser and interacting with the user.
  3. The window object also encompasses propteries releted to the browser environment ,such as the screen dimensions,history and more.
    Document
  1. The document object represents the HTML content of the current web page.It is a property of the global window object.
  2. It provides an interface to access and manipulation the elements content and structure of the web page.
  3. The document object essentially a representation of the entire HTML document , including all its elements,text and attributes.
